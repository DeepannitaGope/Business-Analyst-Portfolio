- Annual Region-wise Claims
- Region & Product category-wise premium
- Product Category-wise Fraud
- Gender-wise Claims in each Category
- Gender-wise Premium in each Category(with Highest highlighted)
- Monthly Claim Graph
- Average Settlement Duration (From Claim to Settlement)
